### ASCII 로 표현 가능한 값 찾기
> - "str[i] & 0x80" 을 조건식으로 넣어주면 된다.    
> - 0x80 = 1000 0000 이다. 그리고 아스키 코드 값은 0000 0000 ~ 0111 1111 이다.     
> - 이를 이용하여 한글, 한자 등을 입력 못하게 할 수 있다.
